# explosao-hn-payloads
Repositório de payloads para testes em sistemas homebrew
## Estrutura
- `/payloads` → Scripts e cargas úteis
- `/docs` → Documentação de cada payload
- `/tools` → Scripts de suporte e utilitários
- `/config` → Configurações de rede

## Segurança
Este repositório é para uso **estritamente educacional e testes em ambientes controlados**. Nunca execute payloads em redes ou sistemas de produção sem autorização.

```bash
pip install -r requirements.txt
